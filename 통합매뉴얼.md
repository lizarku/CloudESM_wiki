이 통합 매뉴얼은 CloudESM SOAR 플랫폼의 모든 문서를 체계적으로 정리한 **중앙 허브(Hub)** 역할을 합니다.

### 문서 구조

본 매뉴얼은 다음 **5개 카테고리**로 구성되어 있습니다:

```
통합매뉴얼.md (현재 문서)
│
├─ 1. 환경설정.md      → 시스템 설치, 구축, 초기 설정
├─ 2. 운영관리.md      → 일상 운영, 모니터링, 유지보수

└─ 5. 개선사항.md      → TODO, 향후 계획, 개선 포인트
```

## 🔍 빠른 참조

### 자주 사용하는 명령어

#### 서비스 관리
```bash
# 전체 서비스 시작
cd /CloudESM/app && ./start.sh

# 전체 서비스 중지
cd /CloudESM/app && ./stop.sh

# Tomcat 재시작
cd /CloudESM/app/tomcat9/bin
./shutdown.sh && sleep 3 && ./startup.sh
```

#### 로그 확인
```bash
# 실시간 로그 확인
tail -f /CloudESM/logs/SearchEngine.log
tail -f /CloudESM/logs/AnalyzeEngine.log
tail -f /CloudESM/logs/catalina.out

# 에러 로그 확인
ls -lh /CloudESM/logs/.error/
```

#### 데이터베이스
```bash
# MariaDB 접속
mysql -u root -p
# 비밀번호: admin@123

# 특정 DB 접속 (로컬)
mysql -u root -p CloudESM
```

### 주요 디렉토리

| 경로 | 설명 |
|------|------|
| `/CloudESM/app/` | 애플리케이션 서비스 |
| `/CloudESM/logs/` | 중앙 로그 디렉토리 |
| `/CloudESM/data/` | 런타임 데이터 |
| `/CloudESM/store/` | 백업 및 정책 데이터 |
| `/CloudESM/app/www/ROOT/` | 웹 애플리케이션 |

### 주요 포트

| 서비스 | 포트 | 설명 |
|--------|------|------|
| Tomcat HTTP | 8080 | 웹 서버 |
| Tomcat HTTPS | 8443 | 보안 웹 서버 |
| MariaDB | 13306 | 데이터베이스 |
| Searcher | 9092 | 검색 엔진 |
